## LISY

LISY is a Windows desktop app for listing, backing-up, and viewing a folder.

## Files

- lisy.go : Go code
- Lisy.exe : Windows executable file built by using lisy.go
- go.mod and go.sum : Files generated upon running 'go mod tidy'
